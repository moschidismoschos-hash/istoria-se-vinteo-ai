ΕΝΗΜΕΡΩΣΗ 8 — ΔΙΚΕΣ ΜΟΥ ΦΩΤΟΓΡΑΦΙΕΣ

Τι προστέθηκε:
- Κουμπί επιλογής φωτογραφιών μέσα στην εφαρμογή.
- Υποστήριξη από 1 έως 20 φωτογραφιών.
- Επιτρεπόμενες μορφές: JPG, JPEG, PNG και WEBP.
- Οι φωτογραφίες αντιστοιχίζονται στις σκηνές με τη σειρά.
- Αν είναι λιγότερες από τις σκηνές, επαναλαμβάνονται αυτόματα.
- Όλες οι φωτογραφίες μετατρέπονται σε ίδιο μέγεθος πριν από τη δημιουργία.
- Συνεχίζει να λειτουργεί η ελληνική αφήγηση.
- Αν δεν επιλεγούν φωτογραφίες, χρησιμοποιούνται οι τρεις δοκιμαστικές.

Δεν χρειάζεται τόκεν και δεν υπάρχει χρέωση.

Εγκατάσταση:
unzip -o ~/storage/downloads/enimerosi-istoria-v8.zip
pip install -r requirements.txt
python app.py
