# Ιστορία σε Βίντεο ΑΙ

Πρώτη λειτουργική έκδοση της εφαρμογής για κινητό.

## Εκτέλεση στο Τέρμουξ

```bash
pkg update -y
pkg install python git -y
python -m pip install --upgrade pip
pip install -r requirements.txt
python app.py
```

Άνοιξε στον περιηγητή:

```text
http://127.0.0.1:5000
```

Η πρώτη έκδοση περιλαμβάνει την τελική εμφάνιση, τις επιλογές, την αποστολή της ιστορίας στον τοπικό εξυπηρετητή και τη δοκιμαστική ένδειξη προόδου. Η πραγματική δημιουργία βίντεο θα συνδεθεί σε επόμενο στάδιο μέσω υπηρεσίας δημιουργίας βίντεο.
