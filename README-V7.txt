ΕΝΗΜΕΡΩΣΗ 7 — ΕΛΛΗΝΙΚΗ ΑΦΗΓΗΣΗ

Τι προστέθηκε:
- Δημιουργία ελληνικής αφήγησης από το κείμενο της ιστορίας.
- Γυναικεία φωνή: el-GR-AthinaNeural.
- Ανδρική φωνή: el-GR-NestorasNeural.
- Ένωση της αφήγησης με το δοκιμαστικό βίντεο μέσω ffmpeg.
- Η επιλογή «Χωρίς αφήγηση» συνεχίζει να λειτουργεί.
- Αν αποτύχει το διαδίκτυο ή η αφήγηση, παραμένει διαθέσιμο το βίντεο χωρίς ήχο.

Εγκατάσταση:
pip install -r requirements.txt
python app.py
